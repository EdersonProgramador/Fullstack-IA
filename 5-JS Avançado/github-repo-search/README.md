# Projeto: Buscador de Repositórios no GitHub

## Descrição

Este projeto é um buscador de repositórios no GitHub. O usuário pode digitar um termo de pesquisa e, ao pressionar ENTER, a aplicação faz uma busca na API do GitHub e exibe os resultados encontrados.

## Funcionalidades

- Busca repositórios no GitHub com base em um termo fornecido pelo usuário.
- Exibe informações relevantes sobre cada repositório, incluindo:
  - Nome e foto do autor
  - Nome do repositório
  - Descrição do repositório
  - Linguagem principal utilizada
  - Link direto para o repositório
- Exibe um estado de carregamento enquanto a busca está em andamento.
- Mostra uma mensagem amigável em caso de erro ou se não houver resultados.

## Estrutura do Projeto

O projeto é composto pelos seguintes arquivos:

- `index.html`: Estrutura principal da aplicação, contendo o campo de busca e a área de resultados.
- `styles.css`: Estilos da aplicação, incluindo layout e design dos cards.
- `script.js`: Lógica da aplicação, gerenciando a busca e a exibição dos resultados.

## Como Executar

1. Clone o repositório ou baixe os arquivos do projeto.
2. Abra o arquivo `index.html` em um navegador web.
3. Digite um termo de pesquisa no campo de busca e pressione ENTER para ver os resultados.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- API do GitHub
